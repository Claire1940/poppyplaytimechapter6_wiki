## ロリー・レッドケープのロア（背景設定）

## ステージの準備：戦闘前のパズル

戦闘が始まる前に、ロリー・レッドケープはプレイヤーに一連の環境タスクを強います。彼は「悲劇には目撃者」と適切な舞台装置が必要だと信じています。彼の指示に従わない場合、報復としてプレイヤーを「台無し」にし、即座にゲームオーバーとなります。

| タスクカテゴリー | 必要なアクション | 重要な注意点 |
| :--- | :--- | :--- |
| **グラブパック** | 背面の壁からパックを回収する | 以降のすべてのステップで必須 |
| **カットアウト** | キャラクターの切り抜きパネルを指定のスロットに配置する | ロリーを満足させるには正確さが必要 |
| **雰囲気作り** | 屋根裏から太陽と雲を下ろす | グラブパックを使ってチェーンを引く |
| **観客** | おもちゃの観客を集めて最前列に配置する | 「照明」が上がる前に素早く行う必要がある |

> **警告：** 準備フェーズ中にロリーを攻撃しようとしないでください。カーテンが降りている間、彼は無敵であり、いかなる攻撃もスクリプト化された死亡シーケンスにつながります。

## ボス戦フェーズ1：モンスターと塔

ステージが整うと、ロリーは「モンスター」を紹介します。これは「プリンセス」が捕らえられている中央の塔を守る、巨大で毛むくじゃらの獣です。このフェーズは俊敏性とリソース管理のテストです。ロリーが傍観者としてあなたの死のナレーターを務め、嘲笑する中で、ステージをナビゲートしなければなりません。

### モンスター戦のメカニクス

モンスターは圧倒的なパワーを持っていますが、スピードに欠けます。目標は獣を直接殺すことではなく、ロリーが不注意にも弱めてしまった、モンスターをステージに繋ぎ止めているチェーンを解体することです。

1.  **回避と誘導：** モンスターは広範囲のなぎ払い攻撃を繰り出します。ステージの端に留まって攻撃を誘い出し、その後内側に移動してください。
2.  **弱点を見つける：** モンスターの腕にある、光っている損傷したチェーンを探してください。これらはロリーが独白の中で言及する「弱い環（Weak links）」です。
3.  **属性の相互作用：** チャプター6では、グラブパックがアイス（氷）とファイア（炎）モジュールでアップグレードされます。これらを切り替えてチェーンを攻撃する必要があります。
4.  **パックの充電：** おもちゃの観客の後ろに2つの充電ステーションがあります。構造的な支柱に十分なダメージを与えるために、パックの充電を維持する必要があります。

| 攻撃タイプ | ダメージレベル | 対処法 |
| :--- | :--- | :--- |
| **オーバーヘッドスラム** | 高（即死） | 影が現れたら横方向にダッシュ |
| **ヒッシングスワイプ** | 中 | 高さに応じてジャンプまたはしゃがむ |
| **チェーンラトル** | 低（スタン） | アイスモジュールを使用してチェーンの動きを凍らせる |

## ボス戦フェーズ2：ロリーの変貌

モンスターが無力化されると、ロリー・レッドケープは冷静さを失います。ショーは台本通りに進まなくなり、彼は自らプレイヤーを始末するために屋根裏から降りてきます。ここで、人気のゲームプレイ動画のタイトル「ロリーがモンスターに変身する（Rory Turns Into a Monster）」が現実のものとなります。彼はステージの機械装置と融合し、布地、ワイヤー、演劇の小道具が入り混じった巨大な塊へと姿を変えます。

### 最終決戦の戦略

このフェーズでは、環境が主要な武器となります。ロリーは吊るされた雲や太陽をフレイル（鎖鉄球）のように使って攻撃してきます。

*   **熱管理：** ロリーが「可愛いシールド」を使用している間は、グラブパックのファイアモジュールを使用して、彼のアーマーの金属部品を過熱させる必要があります。
*   **弱点：** ロリーの顔は回転するマスクの後ろに隠されています。「泣き顔」のマスクが見えている時だけダメージを与えることができます。
*   **環境ハザード：** ステージの照明に注意してください。赤色に変わった場合、そのセクションの床に電気が流れる合図です。

**Q: ロリー・レッドケープ戦での「未完成のシーン（Unfinished Scene）」エラーはどうすれば直りますか？**

**Q: 第1フェーズでモンスターを救うことはできますか？**

A: いいえ。モンスターはストーリーを進行させるために打ち倒されるべき悲劇的な存在です。モンスターを「救う」ことに集中すると、ロリーが「台本を壊した」として直接あなたを攻撃してきます。

**Q: ロリー・レッドケープ戦で最適なグラブパックのモジュールは何ですか？**

A: 両方使う必要があります。フェーズ1では**アイス**を使用して動くチェーンを安定させ、フェーズ2では**ファイア**を使用してロリーの金属シールドにダメージを与えます。素早く切り替えることが、高ランクでのクリアの鍵となります。
